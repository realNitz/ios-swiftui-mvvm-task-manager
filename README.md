# Task Manager iOS App (SwiftUI)

A simple **Task Manager iOS application** built using **SwiftUI** that
demonstrates modern iOS development practices such as MVVM architecture,
async/await networking, pull‑to‑refresh, and sheet‑based editing.

## Features

-   View list of tasks
-   Pull to refresh tasks
-   Add new task
-   Edit task using **SwiftUI sheet-based editing flow**
-   Error handling and loading state
-   Clean **MVVM architecture**
-   Async/Await networking
-   SwiftUI based UI
